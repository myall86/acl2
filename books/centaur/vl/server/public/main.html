Module Browser

This is the default main page. You can customize this page by copying "centaur/vl/server/public" to a new directory, replacing this main.html with your customized version, and supplying "--public=your-public-dir" as an argument to "vl server".

Missing Features from version 2.0

  • Pictures of the module hierarchy (requires calling out to dot)
  • Git blame in the browser (requires calling out to git)
  • Links to github, current hash, etc.
  • Comment system (silly, scrap it?)

Known Problems and Wishlist

  • A garbage collection can take the server a full minute, and the user has no idea that this is happening. It may look like your browser is waiting for a page.
  • It would be nice to have a hierarchical way of navigating the design, i.e., top-level modules first.
